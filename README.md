# Проект приложение для тренировки реакции
## Общее описание
Максимально простое приложение: появляются точки на экране, задача пользователя - как можно быстрее на них нажимать.

## Цели:
- Попробовать работу в Android Studio
- Практика в написании мобильных приложений
- Создать тренажер, который, *возможно*, поможет улучшить реакцию, что поможет развитию в автоспорте...

## Технологии:
- Android Studio
- Java
